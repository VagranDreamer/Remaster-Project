DAO Modding: Correct installation of the Toolset and fix of some errors.


  • 1.The game must be installed before installing the Toolset.
  • 2.The toolset requires Microsoft SQL Server 2005 which comes with it. Uninstall other versions if they are available on your PC, install the correct version and this will save you from half of the problems you may encounter.
  • 3.Sometimes something is treated with a simple reinstallation, you should remember this. Next, we will analyze a common mistake.
  • 4. "origins game install not found" Press on the keyboard Win + R -> enter in the line regedit.exe press "Enter". The registry editor will open. We need to create a path to the game at this address: _LOCAL_MACHINE\SOFTWARE\Wow6432Node\BioWare\Dragon Age
  • To do this, go to THE_LOCAL_MACHINE\SOFTWARE\ and right-click on "Wow6432Node", click "new" -> "key" call it "BioWare" then create "Dragon Age" in the same way how you created "BioWare"
  • Next, click on "Dragon Age" and in the window on the right, right-click in an empty place "new" -> "string value" call it "Path" Right-click on "Path" and click "Modify". And here you insert the path where you have the game installed, for example - C:\Games\Dragon Age Origins
  • Sometimes it happens that "Path" already exists and is has name "Patch_" rename it with the removal of the underscore _ after which the Toolset is successfully installed!
  • 5.You can also familiarize yourself with these materials.
