💡INSTALLATION AND REQUIREMENTS



    Requirements:

  • - You need a clean(without mods and reshade) Witcher 3 version 1.31/1.32 with all DLC.
  • - Put Mod limit executable patcher.exe in the game's bin/x64 directory and run it.
  • - Move the folders from the archive to the folder with the game and confirm the replacement of files.
  • - Download the HDReworkedProject mod extract the folder from the archive and rename parts of mod to: mod_15c_HDReworkedProject1, mod_15c_HDReworkedProject2, mod_15c_HDReworkedProject3, mod_15c_HDReworkedProject4 then place it in the mods folder.
  • - Transfer the configuration files to the Witcher 3 documents.
  • - Launch the game, go to mods and additional content settings and turn on everything as in the screenshots of the settings. Then start your adventure.

    Recommendations:

  • - Graphics card GTX 1070 and above
  • - Do not run other background applications, disable everything including the browser
  • - Give TW3 the highest priority in the process flow
  • - Do not use heavy Reshade presets
  • - Do not load the assembly with a large number of new 3d models or lods, this will lead to fps instability
  • - Not using two textures of Geralt's face or eyes at the same time can lead to endless loading.
  • - Not using a large number of modifications leads to endless loading.

    The assembly consists of the following parts:

  • 1.Core pack (the main package that is mandatory for all users)
  • 2.Optional pack (the content is selectively placed by the user in the mods folder or replaces the existing files directly)

    Reminder:

  • 1.The Witcher mods are installed by moving the unzipped folders to the mods folder. C:\Games\The Witcher 3\mods
  • If there is no such folder, launch the game once and it will appear or create it yourself.
  • 2.The mods at the top of the list overwrite the mods below. If the mod does not work, it is recommended to move it higher.
  • 3.Some mods have required configuration files, they are moved to the bin folder.
  • 4.There are mods that require you to replace your settings of the game itself.
  • 5.After installing a third-party mod that affects scripts you will receive an error message, it is necessary to merge conflicting files using the script merger program.This is only necessary if you have installed your mods, the build already has its own script file, so you do not need to do this.

    ADDITIONAL OPTIMIZATION:

  • 1.For additional optimization, you can remove the following mods: mod_15c_EnvTextures_NovigradReloadedFullPackage, mod_15c_EnvTextures_E3Wall, mod_15c_EnvTextures_BeautifulRocksDark, mod_15b_HDBirds_v11, mod_2b_Graphic_MoreShadows, mod_2b_Graphic_MoreShadows_Toussaint, mod_9_Tweak_NewTorch, mod_2b_Graphic_AQUA, mod_2b_Graphic_NextGenCandles, mod_15c_EnvTextures_UHDOaksRedux_AlternateMoss, mod_15c_EnvTextures_Moon4K, mod_15c_EnvTextures_AlnusNew, mod_2b_Graphic_Snow, mod_1_0_0UltimateClouds, mod_1_0_0Rain.
  • 2.You can delete the new shader.

    Incompatible mods and other observations:

  • 1.Cutscenes Graphics Enhanced --- Disable Fake Lights in All Quest (the game will crash, for example, in a quest to find a friend herbalist)
  • 2.Lip Movement and HiRes Shadows (and High Quality Heads) on NPCs in gameplay --- High Quality Faces (do the same thing, the latter disrupts the work of retextures)
  • 3.Modular Eyes --- BeardGrowthTillStage2 (the eyes arbitrarily change the stage of beard growth and it may look different in scenes).
  • 4.TrissConceptRecolor --- TrissTW2Outfit (texture is superimposed on top)
  • 5.BookMedallion --- ANTRMedallion (texture is superimposed on top)

Questions:



How do I get Geralt's vanilla face back?

- Delete the GeraltTW2style folder or replace it with another appearance mod.

How to replace Geralt's eyes?

- Delete the mod_000_Geralt_TW2_Eyes file and install any other options.

Where can I find new weapons and armor?

- It can be bought in Wyzima, in the baron's castle, from master blacksmith Hattori or created according to the found drawing.

How can I change the appearance of Triss, Yennefer, Keira?

- Delete mods containing Triss, Yennefer, Keira in their name and after that, install your modifications.

Geralt's beard is not getting long is this a bug?

- No, a mod has been installed for the appearance of the beard and a growth stage fixator so that Geralt does not look like Santa, if you want to restore growth, delete these files: mod_8a_Beard_BeardGrowthTillStage2, mod_8a_Beard_Beard2GrowthTillStage2_for_HoSFix

How change Battle System to E3ARP?

- When you installing E3ARP, delete the following files in the game folder: mod_3_Gameplay_SCAARAIOVengeance, dlcSCAARAIOVengeance, modscaar, modscaar.xml, modscaar.xml.bak When all new files will be located where you need to use WitcherScriptMerger to combine scripts.

Is it possible to install other mods on the assembly?

- Yes, but don't install too much, otherwise your game won't load.

Is it possible to use additional mods for LODS?

- Yes, you may have to play with the position of the mods in the list to make it work correctly, but this will significantly reduce your performance.

Hairstyles for women don't work. What's the matter?

- The assembly already has vanilla hair replacers. This is: mod_8_Hairs_zzYenneferBetterHair, mod_8_Hairs_zYenHairTotalBlack, mod_8_Hairs_CiriBetterHair to change the hair you will have to remove it.

Is Hairworks for hair not working for me? Is it a bug?

- No, it is forcibly disabled for hair and enabled for beard. To enable the feature completely delete the mod_8_Hairs_DisableHWHair file.

How do I install a book medallion?

- You need to delete mod_10b_ArmorTextures_ANTRMedallion and install new mod.

Long loose hair and ponytail now look different...

- If you want a vanilla look, order new hairstyles from the barber.

I want to change the lighting mod, what should I remove?

- modWillyBBCedition, dlcWillyBBCedition and mod_ALimaLightingTossaint.

Does blood sometimes appear on my character? Is this a bug?

- Unfortunately, yes, you will have to put up with it or remove the mod for more blood splashes.

Will there be updates?

- Yes, there will be updates as mods for this wonderful game are still coming out, but don't expect frequent updates.
